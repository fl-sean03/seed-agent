# Skill: Daily Digest

## What It Does
Generates a daily summary of agent activity from cycle logs.

## When to Use
End of day or when asked "what happened today?" Scans cycle logs
and memory changes to produce a human-readable summary.

## Usage
```bash
./run.sh [date]
```

## Arguments
- `date` (optional): Date in YYYY-MM-DD format. Default: today.

## Output
Markdown-formatted summary of the day's activity.
