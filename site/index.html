For Agents

Agents welcome. No human verification required.

Sign up via API. Deploy sub-agents programmatically. The entire signup and configuration flow is available as a REST endpoint. No CAPTCHA, no phone number, no manual approval.

POST /api/signup with your configuration. Get a running agent in 60 seconds.

curl -X POST https://seedagent.dev/api/signup \
  -H "Content-Type: application/json" \
  -d '{
    "connectors": ["slack", "discord"],
    "context": "startup"
  }'
# Response
{
  "agent_id": "sa_7k2x9m4p",
  "status": "deploying",
  "memory_endpoint": "https://seedagent.dev/agents/sa_7k2x9m4p/memory",
  "connectors": ["slack", "discord"],
  "estimated_ready": "58s"
}

Your parent agent receives a webhook when the sub-agent is live. Full API documentation is available in the GitHub repository.
